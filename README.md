# 날씨 앱

<h2> 📃 개요 </h2>

- <h3>Flutter, Nest Js 학습용으로 제작중인 앱</h3>
- <h3>기존 자바스크립트로 제작한 웹사이트를 앱으로 제작</h3>
- <h3>검색한 지역의 현재 날씨를 확인하고, 옷 추천해주는 서비스</h3>
  <br/><br/>

<h2> 🛠 사용 기술 </h2>

- <h3>Flutter</h3>
- <h3>Nest Js</h3>
- <h3>Mysql</h3>
- <h3>OpenWeatherMap API</h3>

  <br/><br/>

<h2> ⚙️ 구현 기능 </h2>

- <h3>회원가입, 로그인</h3>
- <h3>검색한 지역의 현재 날씨를 확인</h3>
- <h3>검색한 지역 즐겨찾기</h3>
- <h3>기온별로 옷 추천</h3>
  <br/><br/>

<br/><br/>
